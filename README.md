# FindRecipes

## Project Overview:
Recipe Finder is a web application that allows users to search for recipes based on keywords or filters. Users can view recipe details, and save their favorite recipes.

## Tech stack 

### Frontend
- HTML
- CSS
- JavaScript
- React
- Chakra UI
  
### Backend
- Node.js
- Express.js
- MongoDB
- Spoonacular API
